# PC Wrapped — Charts (category donut + hourly bars) — Design Spec

**Дата:** 2026-06-22
**Статус:** Утверждён (готов к плану)
**Контекст:** Метрики `Aggregator.ByCategory` (время по категориям) и `HourlySeconds` (24 значения) уже считаются, но нигде не визуализируются. Добавляем два графика и на дашборд, и на карточку для шеринга.

## 1. Объём
- Кольцевая диаграмма категорий + столбики активности по часам.
- Размещение: и в главном окне, и на экспортируемой карточке (1:1 и 9:16).
- Вне объёма (YAGNI): легенда с абсолютным временем (только %), интерактив/тултипы на графиках, анимации, выбор палитры.

## 2. Контролы (Avalonia-примитивы, переиспользуемые)
Лежат в `src/PcWrapped/Controls/`. Каждый — `Control` с переопределённым `Render`, работает и в живом окне, и при рендере карточки в `RenderTargetBitmap`.

- **`CategoryDonut`** — кольцо из долей категорий.
  - Вход: `IReadOnlyList<CategorySegment>` (категория, доля 0..1, цвет) — готовится хелпером `ChartData`.
  - Фикс-палитра: Работа `#7B2FF7`, Игры `#F107A3`, Соцсети `#3FB950`, Браузер `#58A6FF`, Прочее `#8A8F9B`.
  - Пустой вход (нет данных) → рисует пустое кольцо фоновым цветом, без падения.
  - Свойства для подгонки: диаметр/толщина кольца (через размеры контрола).

- **`HourlyBars`** — 24 столбика.
  - Вход: `IReadOnlyList<int>` (длина 24) + кисть.
  - Высота столбика = значение / max (если max=0 → все нулевой высоты, без падения/деления на ноль).

## 3. Дашборд (MainWindow)
Новая секция в центральной колонке между шапкой и списком приложений:
- Слева: `CategoryDonut` + легенда (свотч + название + %), только категории с долей > 0, по убыванию.
- Справа: `HourlyBars` с подписями оси 0 / 6 / 12 / 18 / 23.
Список приложений уезжает ниже (в существующем `ScrollViewer`). Секция компактная (высота ~140–160px).

## 4. Карточка (CardRenderer.BuildCard)
- Добавляется компактная полоса: мини-`CategoryDonut` + легенда топ-3 категорий, и мини-`HourlyBars`.
- **Чтобы влезло на 1:1:** на карточке показываем **топ-3 приложения** (вместо текущих топ-5). На 9:16 места достаточно; на 1:1 топ-3 + графики + vanity-строки помещаются.
- Графики опциональны как и иконки: если данных нет — полоса рисуется пустой/скрывается, рендер не падает. Headless-тест остаётся зелёным.

## 5. Данные / хелпер
Чистый хелпер `ChartData` (в `PcWrapped.Core` или app — без UI-зависимостей; цвета держим в app-слое, в Core только доли):
- `Segments(IReadOnlyDictionary<Category, TimeSpan>) → IReadOnlyList<(Category, double fraction)>` — доли от суммы, по убыванию, пустой вход → пустой список.
- `NormalizeHours(IReadOnlyList<int>) → IReadOnlyList<double>` — каждое значение / max (max=0 → все 0).
Палитра `Category → Color` — в app-слое (контролы рисуют по ней).

## 6. Тестирование
- Юнит (Core): `ChartData.Segments` (суммы долей = 1 при непустом входе, порядок по убыванию, пустой → пустой), `NormalizeHours` (нормализация, all-zero → все 0, длина сохраняется).
- Headless (App.Tests): карточка с графиками рендерится в валидный PNG для всех тем; рендер с пустыми `ByCategory`/`HourlySeconds` не падает.
- Сами контролы (рисование) — платформенно, проверяются ручным запуском + headless-рендером карточки.
